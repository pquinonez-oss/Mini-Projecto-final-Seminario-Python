# Mini-Proyecto-final-Seminario-Python
Seminario Python Intr. Linguistas y traductores UOC
Este programa fue realizado con ayuda de la inteligencia artificial ChatGpt
Este programa es una herramienta para el Procesamiento del Lenguaje Natural (PLN) para analizar textos en español. La interfaz permite cargar textos, los analiza automáticamente y se obtienen estadísticas como lemas, categoría gramatical, dependencias sintácticas etc.
Finalmente, permire guardar los resultadso en txt.
